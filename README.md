Temporizador de 2:30 para las partidas del robot de los ebentos FIRST LEGO LEAGUE Challenge. Tiene sonidos en español incluidos.

Se puede poner en local sin servidor.

Para usarlo, abrir en un navegador independiente. Tener durante las partidas la tabla de clasificación del evento, y encima una pequeña ventana con el botón "Temporizador".
Para arrancar el temporizador, pulsar en el botón antes del inicio de la partida y la pantalla se maximizará. 
Cuando se quiera iniciar la partida pulsar sobre "FIRST LEGO LEAGUE". 
30 segundos despues de acabar la partida, la ventana se restaurá al tamaño pequeño.
